## Features
- User and Admin Can login.
- Analysis on the basis of sales(Automatic).
- Admin Can Add or Delete products.
- Logout Option using Session.
- Stock out option When out of Stock.
- Search and Filter options.

## Installation

Install my-project

Cloning github

```
git clone https://github.com/username/repo-name.git
cd Grocery-booking-App
cd code 
```
Create a virtual environment (replace 'venv' with your preferred name)
```
python -m venv venv
```

Starting the venv

```
cd venv\scripts\activate
cd ../..
```

Installing Dependency.
```
pip install -r requirement.txt

```

Run App
```
python app.py
```
